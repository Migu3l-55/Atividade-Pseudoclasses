Galeria de fotos

NAV: é a tag HTML usada para separar um conjunto de links ou endereços que levam o usuário diretamente às partes mais importantes, do seu interesse ou resumem brevemente o conteúdo da página. Podem levar para outras páginas também.

Formulario: um fieldset contendo 3 campos de input simulando um formulario de contato do usuario ou de assinatura. É pedido o nome do usuário, o e-mail e um checkbox perguntando se ele deve receber novas noticias em seu e-mail.

Explicação: um parágrafo simples que possui a função de explicar sobre sí mesmo (gostou da metalinguagem?)

Galeria: uma pequena galeria de fotos do pinterest (os links diretos para os pins estarão no atributo 'alt' de cada imagem respectivamente além de uma breve descrição de o que cada imagem mostra).
